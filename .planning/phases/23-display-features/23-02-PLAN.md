    **Create `src/features/units/ShowcaseMode.tsx`** with these exact specifications:

    Props interface:
    ```typescript
    interface ShowcaseModeProps {
      units: Unit[];
      photos: Map<number, UnitPhotoWithUrl>;
      factions: Faction[];
      onClose: () => void;
    }
    ```

    Component implementation:
    1. **State:** `const [index, setIndex] = useState(0);`
    2. **Faction lookup memo:** `const factionMap = useMemo(() => new Map(factions.map(f => [f.id, f])), [factions]);`
    3. **Derived values:** `const unit = units[index]`, `const photo = unit ? photos.get(unit.id) : undefined`, `const faction = unit ? factionMap.get(unit.faction_id) : undefined`

    4. **`handleClose` callback** (useCallback, depends on [onClose]):
       - Guard with `isTauri()` from `@tauri-apps/api/core`
       - If Tauri: `await getCurrentWindow().setFullscreen(false)` from `@tauri-apps/api/window`
       - If browser: `if (document.fullscreenElement) await document.exitFullscreen?.()`
       - Then call `onClose()`

    5. **Fullscreen useEffect** (runs once on mount, cleanup on unmount):
       - Mount: `if (isTauri()) await getCurrentWindow().setFullscreen(true)` else `await document.documentElement.requestFullscreen?.()`
       - Cleanup: safety net that calls setFullscreen(false) / exitFullscreen on unmount for any reason
       - Wrap cleanup calls in `.catch(() => {})` to swallow errors during teardown

    6. **Keyboard useEffect** (depends on `[units.length, handleClose]`):
       - `Escape` -> call `handleClose()`
       - `ArrowRight` -> `setIndex((i) => (i + 1) % units.length)`
       - `ArrowLeft` -> `setIndex((i) => (i - 1 + units.length) % units.length)` (Pitfall 6: signed modulo fix)
       - Add `keydown` listener on `document`, remove on cleanup

    7. **Navigation handlers:**
       - `handlePrev = () => setIndex((i) => (i - 1 + units.length) % units.length)`
       - `handleNext = () => setIndex((i) => (i + 1) % units.length)`

    8. **JSX structure** — Root: `<div className="fixed inset-0 z-50 bg-black flex items-center justify-center" data-testid="showcase-overlay">`:

       a. **Exit button** (top-right): `<button className="absolute top-4 right-4 text-white/60 hover:text-white transition-colors z-10" onClick={handleClose} aria-label="Exit Showcase">` containing `<X className="h-6 w-6" />` from lucide-react

       b. **Prev arrow** (left edge): `<button className="absolute left-4 top-1/2 -translate-y-1/2 text-white/40 hover:text-white/80 transition-colors z-10" onClick={handlePrev} aria-label="Previous unit">` containing `<ChevronLeft className="h-8 w-8" />`

       c. **Next arrow** (right edge): same pattern with `right-4`, `ChevronRight`, `aria-label="Next unit"`

       d. **Photo** (centered): `<img src={photo?.assetUrl} alt={unit?.name ?? "Unit photo"} className="max-h-[90vh] max-w-[90vw] object-contain" />`

       e. **Bottom overlay bar**: `<div className="absolute bottom-0 left-0 right-0 bg-gradient-to-t from-black/80 to-transparent px-8 py-6">`:
          - Unit name: `<span className="text-white text-lg font-medium">{unit?.name}</span>`
          - Faction name: `<span className="text-white/60 text-sm ml-3">{faction?.name}</span>`
          - Counter (right-aligned): `<span className="text-white/40 text-sm">{index + 1} of {units.length}</span>`
          - Layout: flex with justify-between, items aligned at bottom
