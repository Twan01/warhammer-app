    Component 1: src/features/recipes/ApplyRecipeDialog.tsx

    Props: ApplyRecipeDialogProps { open: boolean, unitId: number, onClose: () => void }

    State: selectedRecipeId: number | null (useState). When null, show picker. When set, show preview + confirm.

    Picker view: Dialog with Command (cmdk) searchable list. Each CommandItem shows recipe.name + Badge with recipe faction name if present. onSelect sets selectedRecipeId.

    Preview view: When selectedRecipeId is set, load sections via useRecipeSections(selectedRecipeId), steps via useRecipePaints(selectedRecipeId), paints via usePaints(). Build paintMap via useMemo (Map from paint.id to paint). Show:
    - Recipe name as heading
    - If sections.length > 0: SectionedTimeline with steps, sections, paintMap
    - If sections.length === 0: RecipeStepTimeline with steps, paintMap
    - Step count badge: "{steps.length} steps"
    - Back button (resets selectedRecipeId to null)
    - Confirm button: calls useCreateAssignment().mutate({ unit_id: unitId, recipe_id: selectedRecipeId }) with toast.success("Recipe applied.") onSuccess and toast.error onError. onSuccess also calls onClose.

    Reset selectedRecipeId to null when dialog opens (useEffect on open prop).

    Dialog open/close pattern: Dialog open={open} onOpenChange={(o) => { if (!o) onClose(); }}. DialogContent className="p-0 sm:max-w-[480px]".

    Component 2: src/features/units/AppliedRecipesTab.tsx

    Props: AppliedRecipesTabProps { unitId: number, onApplyRecipe: () => void }

    Data: useAssignmentsByUnit(unitId) for assignments. useRecipes() for recipe name lookup (build recipeMap: Map from recipe.id to recipe via useMemo).

    Empty state: When assignments.length === 0 and not loading, show centered column with ClipboardList icon (lucide), "No recipes applied yet." text, and Button size="sm" onClick={onApplyRecipe} labeled "Apply Recipe".

    Assignment cards: For each assignment, render a bordered card (border rounded-md p-3) with:
    - Header row: recipe name (from recipeMap) + Trash2 icon button for delete
    - AssignmentChecklist component (gated: only mount when assignment.id is defined)

    Delete: useDeleteAssignment(). Call mutate with { id: assignment.id, unitId: assignment.unit_id, recipeId: assignment.recipe_id } (all 3 fields required per P2 pitfall). Toast success/error.

    Root div: className="flex flex-col gap-4 p-4" aria-busy={isLoading}.

    Component 3: Modify src/features/units/UnitDetailSheet.tsx

    Add imports: AppliedRecipesTab from "./AppliedRecipesTab", ApplyRecipeDialog from "@/features/recipes/ApplyRecipeDialog".

    Add state: const [applyDialogOpen, setApplyDialogOpen] = useState(false).

    Add 4th TabsTrigger after "journal": TabsTrigger value="recipes" with text "Recipes".

    Add 4th TabsContent after journal TabsContent: TabsContent value="recipes" containing AppliedRecipesTab with unitId={unit.id} and onApplyRecipe={() => setApplyDialogOpen(true)}.

    Render ApplyRecipeDialog as SIBLING to Sheet (NOT inside SheetContent -- P6 pitfall). Place after closing Sheet tag: ApplyRecipeDialog open={applyDialogOpen} unitId={unit.id} onClose={() => setApplyDialogOpen(false)}.
