Per D-03, replace the `getRecipePaintsByRecipe` query (lines 6-8) to add a LEFT JOIN on `recipe_sections` and section-aware ORDER BY:
- Change `SELECT * FROM recipe_steps WHERE recipe_id = $1 ORDER BY order_index ASC` to a multi-line template literal
- Use `SELECT rs.* FROM recipe_steps rs LEFT JOIN recipe_sections s ON s.id = rs.section_id WHERE rs.recipe_id = $1 ORDER BY COALESCE(s.order_index, 999999) ASC, rs.order_index ASC`
- The COALESCE sentinel 999999 handles legacy unsectioned steps (NULL section_id) by sorting them last instead of first (SQLite sorts NULLs first in ASC)
- Keep the return type as `Promise<RecipeStep[]>` — `rs.*` produces the same columns as `*`

Add a new test group in `tests/painting/recipeSections.test.ts` asserting the SQL behavior:
- Mock `getDb` and capture the SQL string passed to `db.select`
