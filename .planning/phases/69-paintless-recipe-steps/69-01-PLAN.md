<objective>
Enable recipe steps to exist without a paint selection. A table-rebuild migration makes paint_id nullable, the save guard is removed, the type system is updated, and the SectionedTimeline availability computation is fixed to exclude null paint_id from counts.

Purpose: Users often have recipe steps like "varnish coat" or "assembly prep" that have no associated paint. Currently these steps are silently discarded on save because of a null guard.
Output: Migration 022, updated RecipeStep type, guard-free save path, null-safe availability computation.
</objective>
