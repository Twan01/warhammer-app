<objective>
Wire the atomic "Done + Log Session" action into Painting Mode with a lightweight session logger Sheet, while preserving the standalone "Mark Done" as the primary fast-path action.

Purpose: Users can optionally record time and notes when completing a step, or skip logging entirely for quick completions. The session logger Sheet is pre-filled from painting mode context so the user only touches two optional fields.

Output: New PaintingSessionSheet component + paintingSessionSchema, modified StepFocalView with button pair, modified PaintingModePage with sheet state and combined handler.
</objective>
