    4. Render the outer container: `w-[280px] border-r border-border bg-card overflow-y-auto` with `p-4` internal padding per UI spec. Wrap step list area in a `div` with `flex flex-col gap-1`.

Badge secondary variant showing "completed/total" count from sectionProgressMap.
    6. Current section gets additional classes: `border-l-3 border-primary bg-accent/50` per D-07, UI spec.
    7. Optional sections (section.optional === 1) get an "Optional" Badge outline variant, text-xs per D-09.
    8. CollapsibleContent renders step sub-items. Each step: `pl-6 py-1.5 flex items-center gap-2 cursor-pointer hover:bg-accent/30 rounded` per UI spec. Click calls goToStep(step.id). Step indicator:
       - Completed: Check icon (h-4 w-4, text-green-500) + step name in text-sm text-muted-foreground with line-through
       - Current (step.id === currentStepId): filled primary dot (h-2 w-2 rounded-full bg-primary) + step name in text-sm font-semibold
       - Pending: hollow muted dot (h-2 w-2 rounded-full border border-muted-foreground) + step name in text-sm
