    Signature: `export function usePaintingModeState(assignmentId: number, recipeId: number)`.

    Internally compose three existing hooks:
    - `useRecipePaints(recipeId)` for `RecipeStep[]`
    - `useRecipeSections(recipeId)` for `RecipeSection[]`
    - `useStepProgress(assignmentId)` for `StepProgress[]`

    Derived state (all via `useMemo`):
    1. `sectionOrderMap`: `Map<number, number>` from `sections.map(s => [s.id, s.order_index])`.
    2. `orderedSteps`: spread `steps` and sort by `[sectionOrderMap.get(step.section_id) ?? 999999, step.order_index]`. Per D-03 and D-04, null section_id steps sort last via 999999 fallback.
    3. `completedSet`: `new Set(progressRows.filter(p => p.completed === 1).map(p => p.recipe_step_id))`. Read `completed` as integer `=== 1`, not truthy.
    4. `initialStepId`: `orderedSteps.find(s => !completedSet.has(s.id))?.id ?? orderedSteps[orderedSteps.length - 1]?.id ?? null`. Use empty dependency array `[]` with `// eslint-disable-next-line react-hooks/exhaustive-deps` comment per Pitfall 4 — mount-only, navigation is controlled state thereafter.

    Navigation state via `useState(initialStepId)`:
    - `currentStepId` state with setter
    - `currentIndex = orderedSteps.findIndex(s => s.id === currentStepId)`
    - `canGoPrev = currentIndex > 0`
    - `canGoNext = currentIndex < orderedSteps.length - 1`
    - `goPrev`: if `currentIndex > 0`, set to `orderedSteps[currentIndex - 1].id`
    - `goNext`: if `currentIndex < orderedSteps.length - 1`, set to `orderedSteps[currentIndex + 1].id`
    - `goToStep(id: number)`: set `currentStepId` to `id`

    Section progress via `useMemo`:
    - `sectionProgressMap`: `Map<number, { completed: number; total: number; name: string }>`. Iterate `orderedSteps`, group by `section_id`, count total and how many are in `completedSet`. Key is section id, skip null section_id entries (they get their own "Unsectioned" entry if any exist).

    Return object: `{ orderedSteps, currentStepId, currentIndex, completedSet, isLoading, canGoPrev, canGoNext, goPrev, goNext, goToStep, sectionProgressMap }`.

    `isLoading` is `stepsQuery.isLoading || sectionsQuery.isLoading || progressQuery.isLoading`.
