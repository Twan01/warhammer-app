**1. Create `src/features/game-day/gameDayStore.ts`:**
- Import `create` from `zustand` and `persist` from `zustand/middleware`
- Define `ChecklistItem` interface: `{ id: string; text: string; checked: boolean }`
- Define `GameDayListState` interface: `{ cp: number; prevCp: number | null; startingCp: number; checklistItems: ChecklistItem[]; usedAbilities: string[] }` — usedAbilities as string[] NOT Set (Set is not JSON-serializable)
- Define `GameDayStore` interface with `listStates: Record<string, GameDayListState>` and actions: `setStartingCp(listId: number, cp: number)`, `spendCp(listId: number, cost: number)`, `gainCp(listId: number)`, `undoCp(listId: number)`, `toggleChecklistItem(listId: number, itemId: string)`, `addChecklistItem(listId: number, text: string)`, `resetChecklist(listId: number)`, `toggleAbilityUsed(listId: number, abilityKey: string)`
- Define `DEFAULT_CHECKLIST` array with 5 items: "Verify army list points" (id: "default-1"), "Check detachment rules" (id: "default-2"), "Review stratagems" (id: "default-3"), "Confirm faction rules" (id: "default-4"), "Set up terrain" (id: "default-5") — all with `checked: false`
- Create helper `getListState(state, listId)` that returns `state.listStates[String(listId)]` or default state `{ cp: 0, prevCp: null, startingCp: 0, checklistItems: [...DEFAULT_CHECKLIST], usedAbilities: [] }`
- Create helper `setListState(state, listId, partial)` that merges partial into list state and returns new `listStates` object
- Export `useGameDayStore = create<GameDayStore>()(persist((set, get) => ({ ... }), { name: "game-day-state" }))` 
- `spendCp`: `const cur = getListState(get(), listId); set({ listStates: { ...get().listStates, [String(listId)]: { ...cur, prevCp: cur.cp, cp: Math.max(0, cur.cp - cost) } } })`
- `gainCp`: similar but `cp: cur.cp + 1`, set `prevCp: cur.cp`
- `undoCp`: restore `cp: cur.prevCp ?? cur.cp`, set `prevCp: null`
- `setStartingCp`: set both `startingCp` and `cp` to the value, reset `prevCp: null`
- `toggleChecklistItem`: flip `checked` boolean on matching item by id
- `addChecklistItem`: push new item with `id: crypto.randomUUID()`, `checked: false`
- `resetChecklist`: set all items' `checked` to `false`
- `toggleAbilityUsed`: if key exists in `usedAbilities`, remove it; else add it
- Export a convenience selector hook: `export function useGameDayListState(listId: number)` that returns `useGameDayStore(s => s.listStates[String(listId)]) ?? { cp: 0, prevCp: null, startingCp: 0, checklistItems: DEFAULT_CHECKLIST, usedAbilities: [] }`
