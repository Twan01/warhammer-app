    Component 1: src/features/recipes/ApplyToUnitsDialog.tsx

    Props: ApplyToUnitsDialogProps { open: boolean, recipe: Recipe (the recipe being applied, with id and name), onClose: () => void }

    Data loading:
    - useUnits() for full unit list
    - useFactions() for faction name lookup (build factionMap via useMemo)
    - useAssignmentsByRecipe(recipe.id) for existing assignments
    - Derive assignedUnitIds: Set via useMemo from existingAssignments.map(a => a.unit_id)

    Multi-select state: selectedUnitIds as Set<number> via useState. toggleUnit function creates new Set copy, adds or deletes id. Reset selectedUnitIds to empty Set when dialog opens (useEffect on open).

    Picker UI: Dialog with Command searchable list. Each CommandItem shows:
    - Checkbox checked={selectedUnitIds.has(unit.id)} (visual indicator only, click handled by onSelect)
    - unit.name
    - Badge with faction name from factionMap
    - disabled={assignedUnitIds.has(unit.id)} with opacity-50 class when already assigned
    - onSelect: if not already assigned, call toggleUnit(unit.id)

    Footer: DialogFooter with Cancel button (variant="outline", onClick onClose) and confirm Button:
    - Text: "Apply to {selectedUnitIds.size} unit{selectedUnitIds.size !== 1 ? 's' : ''}"
    - disabled={bulkCreate.isPending || selectedUnitIds.size === 0}
    - onClick: bulkCreate.mutate({ unitIds: Array.from(selectedUnitIds), recipeId: recipe.id }) with toast.success onSuccess ("Recipe applied to N unit(s).") and toast.error onError. onSuccess also calls onClose.

    Dialog wrapper: Dialog open={open} onOpenChange={(o) => { if (!o) onClose(); }}. DialogContent className="sm:max-w-[480px]".

    Component 2: Modify src/features/recipes/RecipeDetailSheet.tsx

    Add import: ApplyToUnitsDialog from "./ApplyToUnitsDialog".

    Add state: const [applyToUnitsOpen, setApplyToUnitsOpen] = useState(false).

    Add button in SheetFooter BEFORE the "Edit Recipe" button: Button variant="outline" onClick={() => setApplyToUnitsOpen(true)} with text "Apply to Unit(s)".

    Render ApplyToUnitsDialog as SIBLING to Sheet (NOT inside SheetContent -- P6 pitfall). Place after closing Sheet tag: ApplyToUnitsDialog open={applyToUnitsOpen} recipe={recipe} onClose={() => setApplyToUnitsOpen(false)}.
