## Tech Debt (non-blocking)

### Phase 25: Design Foundation

- `bg-forge-black` and `bg-panel-elevated` CSS tokens defined in `globals.css` but never applied to any DOM element. These are valid infrastructure tokens registered in `@theme inline` — Tailwind utility classes resolve correctly if applied. They were intended for Phase 26+ consumption but those phases used pre-existing `bg-card`/shadcn defaults instead.

### Phase 28: Collection + Projects

- `useDeleteRecipe` in `src/hooks/useRecipes.ts` does not invalidate `["kanban-enrichment"]`. When a recipe is deleted, the KanbanCard may display a stale recipe name until React Query's `staleTime` triggers a refetch. `useCreateRecipe` and `useUpdateRecipe` correctly invalidate this key.
