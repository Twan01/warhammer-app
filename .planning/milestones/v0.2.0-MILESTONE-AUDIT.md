## Gap Detail: ARMY-02 (closed 2026-05-03)

**Requirement:** "User can add units from their collection to a list and remove them; each unit-in-list shows painting status badge **and assembled status**"

**Gap found by audit:** `status_assembly` was absent from `getArmyListWithUnits` SQL join, `ArmyListUnitRow` type, and render.

**Fix applied (commit `259f3fc`):**
- `src/db/queries/armyLists.ts` — added `u.status_assembly` to SELECT (line 46)
- `src/types/armyList.ts` — added `status_assembly: number` to `ArmyListUnitRow`
- `src/features/army-lists/ArmyListUnitRow.tsx` — added `<Badge variant="outline">Assembled</Badge>` when `status_assembly === 1`, alongside the existing painting status badge in the Status cell

All 16 affected tests remain green post-fix.
