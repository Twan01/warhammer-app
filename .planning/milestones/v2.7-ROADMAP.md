# v2.7 — Points & Army Lists 2.0

**Phases:** 47–50
**Status:** Planned
**Pre-requisite:** v2.6 complete (rules data needed for ruleset-aware lists and points integration)

## Goal

Upgrade Army Lists from points-tracked lists into true game-preparation objects with ruleset awareness, tactical analysis, and print-ready export.

## Phases

- [ ] Phase 47: Army List Schema Extension — Add ruleset/source version, detachment field, game size (Combat Patrol / Incursion / Strike Force / Onslaught / custom), list status (draft / planned / ready / played / archived), support for wishlist/proxy/duplicate unit types in list entries
- [ ] Phase 48: Advanced Points Handling — Integrate imported points with unit point tiers and loadout-specific points, show warnings for outdated/missing points, display points delta after rules re-sync
- [ ] Phase 49: Tactical Tags & List Analysis — Manual tactical role tags per unit in list (anti-tank, screening, fast mover, etc.), soft gap warnings (low anti-tank, few objective holders, too many unpainted), multi-dimension readiness display
- [ ] Phase 50: Export / Print View — Printable army list view: faction, detachment, total points, unit table with points and status, key ability reminders, user notes

## Requirements

### Ruleset-aware lists

- Ruleset/source version
- Points source
- Detachment
- Game size: Combat Patrol, Incursion, Strike Force, Onslaught, custom
- List status: draft, planned, ready, played, archived

### Advanced points handling

- Imported points, unit point tiers, loadout-specific points, manual override
- Warning when points are outdated or missing
- Points delta after sync

### Owned vs wishlist

List entries can be: owned unit, wishlist/planned unit, proxy placeholder, duplicate squad. Show: owned/missing/painted/unpainted/battle-ready points, wishlist cost estimate.

### List readiness

Per list: ownership, assembly, painting, battle-ready readiness, rules-data freshness.

### Tactical tags

Manual tags: anti-tank, anti-infantry, anti-elite, objective holder, screening, fast mover, deep strike, durable, buff piece, melee threat, shooting threat, transport, indirect, action unit, trading piece.

### List analysis

Soft warnings: low anti-tank, low objective control, few fast units, too many unpainted points, too many missing units, outdated points, no rules notes, missing playbook reminders.

### Export / print

Printable view with: list name, faction, detachment, total points, units, points, status/readiness, key reminders, user notes.

## Key risks

- Points source diversity (imported, tiered, loadout, override) creates complex precedence logic
- Tactical tag list must be extensible — 40K meta shifts

## Acceptance criteria

- Existing army lists migrate cleanly
- User can create lists from owned and wishlist units
- Points calculated correctly from imported, tiered, loadout or override data
- Readiness visible at list level
- Tactical gaps visible
- Export/print view available or scaffolded
