<objective>
Wire override markers, points override input, and diff view into PlaybookTab UI.

Purpose: Make overrides visually distinguishable from imported data (OVRD-05), provide a UI entry point for custom points values (OVRD-01), and surface post-sync changes to the user (OVRD-06, OVRD-07). This completes the user-facing features of Phase 46.

**Scope note for OVRD-06:** The Phase 45 snapshot infrastructure (`rules_snapshot.snapshot_data`) stores only `{id, name}` pairs for each table, not full field values (stats, points, keywords, abilities). Therefore the diff view covers datasheet-level changes (added/removed/renamed datasheets) but cannot show per-field value changes (e.g., "points changed from 100 to 110"). This is a known limitation of the current snapshot format. Extending the snapshot to store full field values would be a future enhancement. The diff view fully satisfies OVRD-07 (removed/renamed datasheets) and partially satisfies OVRD-06 (datasheet-level changes only, not per-field stat/keyword changes).

Output: PlaybookTab stat cells show Pencil icon + accent border for overrides with tooltip showing imported value. Points override input lets users enter a custom points value (OVRD-01). useRulesSync computes post-sync diff. PlaybookTab shows collapsible diff section after sync. Toast shows change summary.
</objective>

    function importedStatValue(key: StatKey): number | null {
      // Datasheet model stats are stored as TEXT in rw_datasheet_models
      const model = datasheet?.models?.[0];
      if (!model) return null;
      const raw = (() => {
        switch (key) {
          case "M": return model.M;
          case "T": return model.T;
          case "Sv": return model.Sv;
          case "W": return model.W;
          case "Ld": return model.Ld;
          case "OC": return model.OC;
          default: return null;
        }
      })();
      // coerce TEXT stat to number (strip quotes, +, etc)
      if (raw === null || raw === undefined || raw === "") return null;
      const cleaned = String(raw).replace(/["+]/g, "");
      const n = Number(cleaned);
      return Number.isFinite(n) ? n : null;
    }
