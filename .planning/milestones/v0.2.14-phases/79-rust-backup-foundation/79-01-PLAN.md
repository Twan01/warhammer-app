1. **BackupManifest struct** per D-02/D-08:
   - Derive both `serde::Serialize` and `serde::Deserialize`, plus `Clone`
   - Fields: `app_version: String`, `schema_version: u32`, `created_at: String`, `platform: String`, `db_size_bytes: u64`
   - Make struct and fields `pub` so commands can construct and return it

2. **format_iso8601_now() -> String** helper:
   - Use `time::OffsetDateTime::now_utc()` to get current UTC time
   - Format as ISO 8601 string using `time::format_description::well_known::Rfc3339`
   - Return the formatted string

3. **format_filename_timestamp() -> String** helper:
   - Use `time::OffsetDateTime::now_local()` (fall back to `now_utc()` if local fails) to get local time
   - Format as `YYYY-MM-DD-HHMM` string (e.g., "2026-05-18-1430") for use in safety backup filenames per D-10

4. **vacuum_to_temp(app: &tauri::AppHandle) -> Result<std::path::PathBuf, String>** async helper:
   - Reuse the exact sqlx direct-connection pattern from `backup_database` (lines 583-596): inline `use sqlx::{sqlite::SqliteConnectOptions, ConnectOptions}` and `use std::str::FromStr`
   - Resolve app_data_dir, build db_url pointing to hobbyforge.db
   - Target temp path: `app_data_dir.join("hobbyforge_backup_temp.db")`
   - Remove existing temp file first (guard against prior failed attempt, ignore NotFound)
   - Execute VACUUM INTO with single-quote escaping: `format!("VACUUM INTO '{}'", temp_path.display().to_string().replace('\'', "''"))`
   - Return the temp_path on success
   - Use `.map_err(|e| format!("context: {e}"))` pattern for all errors

5. **create_backup_zip(db_path: &std::path::Path, metadata: &BackupManifest, dest_path: &std::path::Path) -> Result<(), String>** helper:
   - Use `zip::write::ZipWriter` and `zip::write::SimpleFileOptions`
   - Set compression method to `zip::CompressionMethod::Stored` (SQLite data does not compress well per RESEARCH.md Pitfall 3)
   - Add entry "hobbyforge.db": read entire temp db file via `std::fs::read(db_path)`, write to zip
   - Add entry "metadata.json": serialize metadata via `serde_json::to_string_pretty(metadata)`, write to zip
   - Call `zip.finish()` to finalize
   - Use consistent `.map_err(|e| format!("context: {e}"))` error handling

Add necessary imports at the top of the new section (or inline per established pattern): `use std::io::Write` for the ZipWriter write_all calls.
